Publishing a new version for textures files

  1. Open Maya 2017 using Solstice Launcher

  2. Launch Solstice Pipelinizer tool

    Solstice Pipelinizer Button in Solstice Shelf

  3. Open Solstice Publisher by pressing >PUBLISH NEW VERSION< button located in the bottom right corner of Solstice Pipelinizer

    Solstice Pipelinizer Button in Solstice Shelf

    Warning

    When Solstice Publisher is launched, it checks for the latest version by connecting to Artella server This can take time, so be patient. You will see the following dialog in the center of your Maya viewport.

  4. Make sure TEXTURES checkbox is checked.

    IMPORTANT

    You will notice that when you check TEXTURES checkbox, MODEL and SHADING checkboxes are automatically selected. This is complete normal. This means that each time you publish textures files a new model file and shading will be published.

    This is happening because when textures are published, Solstice Publisher automatically updates shader textures with the latest published textures version files.
